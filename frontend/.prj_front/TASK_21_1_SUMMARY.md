## 关键实现细节

### 分页算法

```typescript
// 总页数计算
totalPages = Math.ceil(total / pageSize)

// 当前页的项目数
itemCount = Math.min(pageSize, total - (page - 1) * pageSize)

// 数据提取
startIndex = (page - 1) * pageSize
endIndex = Math.min(startIndex + pageSize, total)
pageData = data.slice(startIndex, endIndex)
```

### 参数验证规则

- 页码必须 >= 1
- 页面大小必须 > 0
- 总项目数必须 >= 0
- 页码不能超过总页数
- 总项目数为 0 时没有有效的页码

### 错误处理

- 无效页码、页面大小、总数时抛出错误
- 超出范围的页码返回空数组或 0 项
